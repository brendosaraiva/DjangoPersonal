Objetivo: criar um sistema web onde dá dicas para pessoas que buscam diferentes
formas de emagrecer, manter corpo, engordar ou ganhar massa corporal.

index: é a página inicial, nela estão contidas informações breves a respeito
da página imc e a página de exercícios, que somente poderão ser acessadas se
o usuário possuir cadastro ou não.

* login: página de acesso exclusivo de usuários cadastrados no sistema.

* cadastro: página de cadastro de usuários.

imc: página que calcula o índice de massa corporal dos usuários e retorna
uma lista de exercícios que este deve fazer de acordo com seu respectivo IMC.

exercícios: página com links âncoras que levam os usuários diretamente as
atividades físicas de acordo com seu IMC.
